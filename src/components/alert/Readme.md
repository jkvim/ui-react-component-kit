** Usage **

```javascript static
    import { Alert } from 'ui-react-component-kit'
```


```jsx
    </Alert>
```


```jsx
    <Alert type={'warning'}>
        Warning Alert
    </Alert>
```

```jsx
    <Alert type={'danger'}>
        Danger Alert
    </Alert>
```
