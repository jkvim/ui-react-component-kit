** Usage **

```javascript static
    import { Fieldset } from 'ui-react-component-kit'
```

** Blank ** 

```jsx
    <Fieldset 
        id={'fieldset-id-1'}
    >
        Hello
    </Fieldset>
```

** Legend **

```jsx
    <Fieldset 
        id={'fieldset-id-2'} 
        legend={'Demo Legend'}
    >
        Hello
    </Fieldset>
```
